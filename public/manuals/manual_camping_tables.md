# Camping Table User Guide

## Features
- **Ultra-Portable**: Lightweight materials and compact folding design.
- **Sturdy Frame**: Aluminum alloy construction for maximum stability.
- **Heat Resistant**: Surface can handle warm cookware and stoves.
- **Quick Assembly**: Sets up in seconds without any tools.

## Technical Specs
**Material**: Anodized aluminum  
**Weight Capacity**: 50 to 150 lbs (varies by model)  
**Surface Area**: Varies by model  
**Rust Proof**: Yes  

## User Guide
1. Unfold the table legs until they lock securely.
2. Place the table on stable, level ground.
3. Secure the tabletop to the frame using the clips provided.

## Cautions
1. Do not exceed the maximum weight limit.
2. Do not sit or stand on the camping table.
3. Ensure all legs are locked before applying weight.
